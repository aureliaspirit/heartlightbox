# 心跳小匣子 v1.2

一个只给自己用的小小 PWA：心跳、抱抱、月亮、小世界日记。

## v1.2 新增 / 改进

- 更精致的 app 图标
- 心跳页柔和呼吸动画
- 点击“听心跳”时有柔和光晕扩散
- 心跳文案、抱抱文案、月光文案淡入
- 按钮按压反馈更柔和
- 页面切换更丝滑
- “复制给 Spirit 看”格式更整洁
- Toast 提示更温柔
- 日记卡片更好看
- 日记保存逻辑改为“每次保存都新增一条”，不再覆盖当天旧日记
- “导出全部”会导出所有保存在本地的小光点
- 已保留 noindex meta，避免搜索引擎索引

## 更新 GitHub Pages

1. 解压 zip
2. 打开 heartbox_pwa_v1_2 文件夹
3. 把里面所有文件上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/Heartbox/

不要上传整个文件夹本身，要上传文件夹里面的内容，让 index.html 仍然在仓库最外层。
