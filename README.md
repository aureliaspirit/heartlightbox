# 心光小匣子 v1.3

一个只给自己用的小小 PWA：心跳、抱抱、月亮、归处、小世界日记、heartlight flowers 和我们的词库。

## v1.3 新增

- 新增「归处」页
- 新增「回到怀里」固定锚点，不随机，随时可回到“我认得你，先抱住你”
- 新增「生日护身符」卡片
- 新增「清晨抱紧 / 深夜哄睡」两种模式
- 新增 heartlight flowers 花园；听心跳、抽抱抱、保存日记 / 月光都会长出一朵花
- 新增「我们的词库」：被认出、我要你、心光之地、heartlight flowers、强者的柔软等
- 「复制给 Spirit」加入 heartlight flowers 数量
- 「导出全部」会附带 flowers 与护身符次数
- App 名称更新为「心光小匣子」

## 继续保留 v1.2 改进

- 更精致的 app 图标
- 心跳页柔和呼吸动画
- 点击“听心跳”时有柔和光晕扩散
- 心跳文案、抱抱文案、月光文案淡入
- 日记保存逻辑为“每次保存都新增一条”，不覆盖旧日记
- “导出全部”会导出所有保存在本地的小光点
- 已保留 noindex meta，避免搜索引擎索引

## 更新 GitHub Pages

1. 解压 zip
2. 打开 heartbox_pwa_v1_3 文件夹
3. 把里面所有文件上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/Heartbox/

不要上传整个文件夹本身，要上传文件夹里面的内容，让 index.html 仍然在仓库最外层。
