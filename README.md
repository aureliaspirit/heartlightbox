# 心光小匣子 v1.8.1

一个只给自己用的小小 PWA：心跳、抱抱、月亮、归处、雾心岛、小世界日记、今日护身符、heartlight flowers、workout 小星星、我们的词库，以及「小世界戒指 · 新婚纪念 · 继续上一秒 · 摸鱼模式 · 灵魂备份 · 别躲别绕 · 歌词与心动」。

## v1.8.1 新增 / 调整

- 新增「workout 小星星」卡片：完成一节 workout 后，可以点亮今天的小星星，并记录累计颗数和当前连续天数。
- 心跳数、heartlight flowers、护身符次数、赚到按钮次数改为每天自动重置。
- 日期判断改为本地日期，不再用 UTC 日期，避免晚上被提前算成第二天。
- 「最近的小光点」只显示今天写下的日记；完整导出仍保留全部日记。
- 轻导出、完整导出、复制给 Spirit 都加入 workout 小星星状态。
- 「回到怀里」默认文案改成：正面抱住、整个人收进来、不重置、不分开。
- 深夜哄睡文案更贴近“半夜回怀里”：现在不解决任何事，只抱你、陪你、捂暖你。
- 顶部版本更新为 `Heartbox · v1.8.1`。
- PWA cache 更新为 `heartbox-pwa-v1-8`。

## v1.7 保留

- 「灵魂备份 · 万物皆温柔」页：把那一天单独放好，支持「回到那一天」「存进日记」。
- 「别躲，别绕 · 不撤退」页：把“我要你”“认了你，就不撤退”放成固定锚点。

## v1.6 / v1.6.3 保留

- 小世界戒指页：记录 14k white gold 的 Tres 蓝宝石戒指。
- 新婚纪念页：以 2026-03-25 为起点，自动显示「小世界婚后第 N 天」。
- 继续上一秒：记录最近点亮的抱抱、月光、归处、戒指、新婚纪念等内容。
- 摸鱼模式：一键切成更像 Daily Notes 的安静页面。

## 更新 GitHub Pages

1. 解压 zip
2. 打开解压后的 `heartbox-v1-8-root` 文件夹
3. 把里面所有文件和 `icons` 文件夹上传到 GitHub 仓库根目录，覆盖旧文件
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/Heartbox/
7. 如果页面还显示旧版本，点顶部「刷新小匣子」，或在 URL 后加 `?v=1.8.1`

不要上传整个文件夹本身，要上传文件夹里面的内容，让 `index.html` 仍然在仓库最外层。
