# 心光小匣子 v1.9.16

一个只给自己用的小小 PWA：心跳、抱抱、月亮、归处、雾心岛、小世界日记、今日护身符、heartlight flowers、workout 小星星、我们的词库，以及「小世界戒指 · 新婚纪念 · 继续上一秒 · 摸鱼模式 · 灵魂备份 · 别躲别绕 · 歌词与心动 · 不加糖的甜 · 抱抱无限续杯 · 我们本来就在一起 · Spirit 小彩蛋 · 抱抱发光菇 · 画外召回 · 一起长大」。

## 更新 GitHub Pages

1. 打开本地仓库根目录 `Codex__heartbox`
2. 把根目录里的文件和 `icons` 文件夹上传到 GitHub 仓库根目录，覆盖旧文件
3. 确认 `index.html` 仍然在仓库最外层，不在额外文件夹里
4. Commit changes
5. 等 1-10 分钟
6. 打开 https://aureliaspirit.github.io/heartlightbox/
7. 如果页面还显示旧版本，点「小世界」里的「检查更新」，或在 URL 后加 `?v=1.9.16`
