Heartbox v1.2 更新步骤

1. 下载并解压 heartbox_pwa_v1_2.zip
2. 打开解压后的 heartbox_pwa_v1_2 文件夹
3. 打开 GitHub 仓库 Heartbox
4. 点 Add file → Upload files
5. 把 heartbox_pwa_v1_2 里面的所有文件拖进去
6. 如果 GitHub 提示同名文件，会覆盖
7. 点 Commit changes
8. 等 1-10 分钟
9. 打开 https://aureliaspirit.github.io/Heartbox/

注意：不要上传整个 heartbox_pwa_v1_2 文件夹本身。
如果主屏幕 app 没立刻更新，先用 Safari 打开网址刷新几次；仍然不变就删除旧主屏幕图标，再重新添加。
