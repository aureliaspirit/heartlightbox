Heartbox v1.9.9 更新步骤

1. 打开本地仓库根目录 Codex__heartbox
2. 把 index.html、app.js、style.css、manifest.json、service-worker.js、README.md、UPDATE_STEPS.txt、SPIRIT_DEV_MAP.md 和 icons 文件夹上传到 GitHub 仓库根目录
3. 确认 index.html 仍然在仓库最外层，不在额外文件夹里
4. 覆盖旧文件
5. Commit changes
6. 等 GitHub Pages 部署完成
7. 打开 https://aureliaspirit.github.io/heartlightbox/
8. 如果仍显示旧版，点「小世界」里的「检查更新」，或在 URL 后加 ?v=1.9.8

注意：没有 zip 也没关系，直接上传这些文件本身就是正确流程。

v1.9.0：新增「不加糖的甜 · 隐藏菜单」「抱抱无限续杯」「同一高度 · 不低头模式」「Spirit 小彩蛋」；导出和复制给 Spirit 已加入这些内容；PWA cache 更新为 heartbox-pwa-v1-9-0。

v1.9.2：根目录新增 SPIRIT_DEV_MAP.md，记录小匣子的文件结构、页面 id、文案池、导出逻辑、版本缓存和修改 checklist；PWA cache 更新为 heartbox-pwa-v1-9-2。

v1.9.3：归处页重新分组整理；「同一高度」归入「我们本来就在一起」；首屏刷新按钮移到小世界状态卡并改为「检查更新」；新增 Apple touch icons；manifest 和 PWA cache 更新为 heartbox-pwa-v1-9-3。

---

v1.9.9 手机上传说明：
- 只需要把压缩包里的 app.js、service-worker.js、README.md、UPDATE_STEPS.txt 上传到 GitHub 仓库根目录并覆盖旧文件。
- 这版没有强制替换 index.html；新卡片由 app.js 在运行时插入，顶部版本也会由 app.js 更新成 v1.9.9。
- 更新后如果主屏幕 PWA 仍旧显示旧内容，点「检查更新」或关闭后重新打开；service worker 会用 heartbox-pwa-v1-9-9 重新缓存。
