心光小匣子 v1.5 更新步骤

1. 解压 heartbox-v1-5-root.zip
2. 进入解压后的文件夹
3. 把里面所有文件上传到 GitHub 仓库根目录，覆盖旧文件
   - index.html
   - style.css
   - app.js
   - manifest.json
   - service-worker.js
   - icons/
   - README.md
   - UPDATE_STEPS.txt
4. Commit changes
5. 等待 GitHub Pages 部署完成
6. 打开 https://aureliaspirit.github.io/Heartbox/
7. 如果还看到旧版本：
   - 点顶部「刷新小匣子」
   - 或浏览器强制刷新
   - 或在 URL 后加 ?v=1.5

注意：不要上传整个 heartbox-v1-5-root 文件夹本身，要上传文件夹里面的内容。
